# CCM (Claimant Mitra — companion app)

Next.js app scaffold aligned with the master portal stack (Pages Router, Tailwind v4, Next 15 / React 19).

## Run locally

From this folder:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

- `npm run dev` — development server
- `npm run build` — production build
- `npm run start` — production server
- `npm run lint` — ESLint
